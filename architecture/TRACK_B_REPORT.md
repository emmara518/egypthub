## Interview Script

```
1. How do you currently get customers?
   (Probe: WhatsApp, TripAdvisor, walk-ins, referrals?)

2. What is your biggest growth problem?
   (Probe: seasonality, competition, marketing cost?)

3. Do you use any tourism platform today?
   (Probe: Booking.com, GetYourGuide, Viator, local?)

4. Would you pay commission for qualified leads?
   (Probe: what if leads convert? what if they don't?)

5. What commission feels acceptable?
   (Probe: 5%? 10%? 15%? Fixed fee?)

6. What would make you stop using WhatsApp and use EgyptHub?
   (Probe: dashboard? analytics? automatic billing?)
```
